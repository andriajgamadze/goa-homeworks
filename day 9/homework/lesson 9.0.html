html

html ეს არის გული და მთავარი რამ რომ შექმნა website იგი არის მთავარინივთი მანქანის ძრავი, რული, ბორბალი და იგი გამოიყენება საიტის გასაკეთებლად მაგრამ css და js

js

javascript არის დამამთავრებელი, იგი ამთავრებს ყველაფერს და ალამაზებს მთლიან საიტს და ამის გარეშე იგი ძალიან უხეში და არა ლამაზი.

css

css ეს არის შესალამაზებელი ენა რასაც იყენებენ javascript ამდე, css და js მხოლოდ გასალამაზებლად გამოიყენება და html-ი ფუნქსიისთვის.
